Формат сохранений версии 1.2.2d частично не совпадает с форматом сохранений версии 1.2.2с. Однако в новой версии можно безопасно использовать старые сохранения, если сохранение было сделано в определенной точке. Порядок действий такой:
1. Играя в версию 1.2.2с перейдите в ночное время суток.
2. Завершите все текущие ивенты, если они есть: разговоры с Гермионой, Снейпом, прочитайте письма, заберите посылки.
3. На экране игры должен отображаться только джинн сидящий за столом.
4. Сохранитесь.
5. Это сохранение можно безопасно использовать в новой версии.

Сохранение в любой другой точке при определенных условиях также может заработать в новой версии, но его работоспособность не гарантируется.
